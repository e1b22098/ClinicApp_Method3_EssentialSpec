# クリーンビルドエラーの簡単な解決方法

## 方法1: cleanをスキップしてビルド（最も簡単）

cleanが失敗する場合は、cleanをスキップして直接ビルドできます：

```powershell
mvn install -DskipTests
```

これでビルドが完了します。

## 方法2: 直接JARを作成して実行

アプリケーションを起動したい場合：

```powershell
# cleanをスキップしてJARを作成
mvn package -DskipTests

# JARファイルを実行
java -jar target/appointment-1.0.0.jar
```

## 方法3: 手動でtargetディレクトリを削除してからビルド

```powershell
# targetディレクトリを削除（エラーを無視）
Remove-Item -Path target -Recurse -Force -ErrorAction SilentlyContinue

# その後、ビルドを実行（cleanは不要）
mvn install -DskipTests
```

## 推奨：まず方法1を試してください

cleanが失敗しても、`mvn install -DskipTests`を実行すればビルドできます。
この方法が最も簡単で確実です。
