# アプリケーション起動方法

## 方法1: JARファイルを直接実行（推奨）

ビルドが成功している場合、JARファイルが既に作成されています。以下のコマンドで起動できます：

```powershell
java -jar target/appointment-1.0.0.jar
```

この方法が最も確実です。

## 方法2: mvn spring-boot:run を使用する場合

`mvn spring-boot:run`が失敗する場合は、まず再コンパイルを試してください：

```powershell
# 再コンパイル
mvn compile

# その後、起動
mvn spring-boot:run
```

## 方法3: IDEから直接実行

EclipseやIntelliJ IDEAを使用している場合：

1. `ClinicAppointmentApplication.java` を開く
2. `main`メソッドを右クリック
3. "Run As" → "Java Application" を選択

## 起動確認

アプリケーションが正常に起動すると、以下のようなメッセージが表示されます：

```
Started ClinicAppointmentApplication in X.XXX seconds
```

その後、ブラウザで以下にアクセス：

```
http://localhost:8080
```

## トラブルシューティング

### ClassNotFoundException が発生する場合

1. JARファイルを再作成：

```powershell
mvn package -DskipTests
java -jar target/appointment-1.0.0.jar
```

### ポート8080が使用中の場合

`application.yml`でポート番号を変更：

```yaml
server:
  port: 8081
```

### データベース接続エラーが発生する場合

1. MySQLが起動しているか確認
2. `application.yml`の接続情報が正しいか確認
3. データベース`clinic_booking_db`が作成されているか確認
